SMKUI

What is this thing?

SMKUI is a web user interface library by The National Gallery of Denmark (SMK). It provides a set of guidelines for aesthetics, design, and frontend code architecture across the gallery's web based projects.

The design

The design principles used in SMKUI is based on a design manual by E-types which was initally developed with print media in mind. SMKUI is interepreting the original design principles to make them applicable in digial and web based media as well.

The code

The code is available from our GitHub repository. Note that the code is made publically available for educational and inspirational purposes only. Read more about the usage rights and copyrights.
